# Citriage

Lists out modules from Jenkins and their current status as it pertains ot CI Triage rotation.

## Installation

Build the gem:
```shell
gem build citriage.gemspec
```

Install the gem:
```shell
gem install {gem_file}
```

## Usage

Simply run:
```shell
ci-triage
```
